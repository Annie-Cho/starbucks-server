# ☕️ starbucks-server
간단한 스타벅스 서버를 구현하였습니다.
<br>
회원가입 시 입력된 핸드폰 번호로 CoolSMS를 통해 인증번호를 전송하고 인증이 완료되면 회원가입을 할 수 있습니다.
가입절차가 모두 끝나면 Nodemailer를 사용하여 환영 메일을 전송합니다.
